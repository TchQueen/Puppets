# Puppets
A Ren'Py visual novel game.

Puppets is a short black-and-white psychological horror game built in the Ren’Py Visual Novel Engine. The game consists of different narrative paths, centring around three respective character perspectives. Featuring a cat, a rat, and a human child, the player plays through each path while avoiding choices that might lead to death.

The game can be found on itch.io: https://maurmischief.itch.io/puppets

Made with Ren'Py.
